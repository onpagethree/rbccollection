콜롬비아 엘 트리운포

로스팅 베르크로스터스
생두 농장 El Triunfo
지역 Colombia, Huila
품종 Ethiopia Landrace
가공 Washed with Doulbe Anaerobic Fermentation
RBC 추천 복숭아 홍차, 산딸기가 연상되는 향긋한 커피입니다. 과일의 향미와 새콤달콤한 맛을 가진 커피를 즐기는 분께 추천드려요.
RBC 레시피 아이스 추출시에도 향긋한 향미를 잘 전달하는데 집중했습니다.
Hot 원두량 16g / 분쇄도 5(EK43기준) / 물온도 96도 / 추출 시간 총 1분 25초(에어로프레스) / 물 240g 투입 후 1분 25초에 추출 종료(30초 블루밍, 물 190g 추가 투입 후 1분에 25초 동안 천천히 프레스),10g 가수
Iced 원두량 16g / 분쇄도 5(EK43기준) / 물 온도 96도 / 추출 시간 총 1분 25초(에어로프레스)/ 물 150g 투입 후 1분 25초에 추출 종료(30초 블루밍, 물 100g 추가 투입 후 1분에 25초 동안 천천히 프레스)
Ethiopia Sidama Arbegona G1

콜롬비아 엘 포르베니 게이샤 워시드

로스팅 트레져스
생두 농장 El Porvenir
지역 Colombia, Qunidio
품종 Geisha
가공 Washed
RBC 추천 레몬그라스차 같이 은은한 향미를 가진 커피입니다. 차분하고 섬세한 향미를 가진 커피를 즐기는 분께 추천드려요.
RBC 레시피 차처럼 부담 없는 농도에서 은은한 향미를 잘 전달하는데 집중했습니다.
Hot 원두량 16g / 분쇄도 5(EK43기준) / 물온도 96도 / 추출 시간 총 1분 25초(에어로프레스) / 물 240g 투입 후 1분 25초에 추출 종료(30초 블루밍, 물 190g 추가 투입 후 1분에 25초 동안 천천히 프레스),10g 가수
Iced 원두량 16g / 분쇄도 5(EK43기준) / 물 온도 96도 / 추출 시간 총 1분 25초(에어로프레스)/ 물 150g 투입 후 1분 25초에 추출 종료(30초 블루밍, 물 100g 추가 투입 후 1분에 25초 동안 천천히 프레스)
