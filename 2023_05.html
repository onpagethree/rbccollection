에티오피아 타라켄 고투

로스팅 나무사이로
생두 농장 Taraken Gotu Family
지역 Ethiopia, Yirgacheffe, Tulise
품종 Curume, Wellisho, Dega
가공 Natural
RBC 추천 꽃다발이 연상되는 향긋한 커피입니다. 복합적인 향과 농밀한 단맛을 가진 커피를 즐기는 분께 추천드려요.
RBC 레시피 향긋한 향미가 잘 전달되는 것과 매장 사용 효율성을 고려하여 설계했습니다. 에어로프레스를 사용하시는 분은 참고하셔도 좋겠습니다.
Hot 원두량 16g / 분쇄도 9(EK43 기준) / 물 온도 96도 / 추출 시간 총 1분 30초(에어로프레스) / 물 50g 투입 후 30초 블루밍, 물 200g 추가 투입 후 1분에 프레스(30초동안 천천히 프레스)1분 30초에 추출 종료
Iced 원두량 16g / 분쇄도 5 / 물 온도 96도 / 추출 시간 총 1분 20초(에어로프레스) / 물 50g 투입 후 30초 블루밍, 물 100g 추가 투입 후 1분에 프레스(20초동안 천천히 프레스) 1분 20초에 추출 종료

리오하 페루

로스팅 파스텔커피웍스
생두 농장 El Barejon(Elver Rioja)
지역 Peru, El Diamante, Huabal
품종 Catuai, Typica, Bourbon
가공 Washed
RBC 추천 감귤 같은 투명한 산미가 느껴지는 커피입니다. 클래식한 향미와 깔끔한 클린컵을 가진 커피를 즐기는 분께 추천드려요.
RBC 레시피 깔끔한 여운의 클린컵이 잘 전달되는 것과 매장 사용 효율성을 고려하여 설계했습니다. 에어로프레스를 사용하시는 분은 참고하셔도 좋겠습니다.
Hot 원두량 16g / 분쇄도 8(EK43 기준) / 물 온도 96도 / 추출 시간(에어로프레스)물 50g 투입 후 30초 블루밍, 물 190g 추가 투입 후 1분에 프레스(30초동안 천천히 프레스) 1분 30초에 추출 종료, 가수 10
Iced 원두량 16g / 분쇄도 5 / 물온도 96도 / 추출 시간 총 1분 20초(에어로프레스) / 물 50g 투입 후 30초 블루밍, 물 100g 추가 투입 후 1분에 프레스(20초동안 천천히 프레스) 1분 20초에 추출 종료
